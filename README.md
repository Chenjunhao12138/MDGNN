MD-GNN: Multimodal Graph Neural Network for DDI Prediction
This repository provides a PyTorch implementation of the MD-GNN framework as described in the paper: "MD-GNN: A Multimodal Graph Neural Network Framework for Comprehensive Drug–Drug Interaction Prediction".
The model integrates three distinct drug modalities—Molecular Structure (SMILES), Pharmacological Text (BioBERT), and Biological Mechanism Paths (HAN)—using a structure-aware attention mechanism and topology-adaptive gating.
